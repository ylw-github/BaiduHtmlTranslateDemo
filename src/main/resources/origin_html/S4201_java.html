There's no need to null test in conjunction with an instanceof test. null is not an instanceof anything, so a null check is redundant.

不合规的代码示例

if (x != null && x instanceof MyClass) { ... }  //不合规的

if (x == null || ! x instanceof MyClass) { ... } //不合规的

兼容的解决方案

if (x instanceof MyClass) { ... }

if (! x instanceof MyClass) { ... }
